# petAdoption

This is an alpha version of the Pet Adoption application. 
Current features include : 
  - Chat system
  - Post viewing and creating
  - Login / sign up
  - post removal
  
Installation:
  - Download the repository.
  - Open it up inside Android Studio.
  - Build it and run
